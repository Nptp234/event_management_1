# Event Mangement

## Yêu cầu hệ thống
- Flutter SDK >= 3.0
- Dart >= 2.18
- Android Studio hoặc Xcode (tuỳ vào nền tảng mục tiêu)
- Visual Studio Code (tùy chọn)

## Cài đặt
### 1. Clone repository
```bash
git clone https://github.com/Nptp234/event_management_1.git
cd event_management_1
```

### 2. Cài đặt các dependencies
```bash
flutter pub get
```

### 3. Build cho Android
``` bash
flutter build apk
```

### 4. Build cho IOS
``` bash
flutter build ios
```

### 5. Kiểm tra cấu hình
``` bash
flutter doctor
```

### 6. Chạy dự án
``` bash
flutter run
```
